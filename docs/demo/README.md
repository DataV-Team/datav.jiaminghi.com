# DEMO

::: tip TIP
Demo均使用了[全屏容器](/guide/fullScreenContainer.html)进行内容缩放，查看时请F11全屏以便于Demo充满屏幕。

Mac的Retina及其他高分辨率屏幕下查看Demo可能出现缩放异常，如出现异常请调节屏幕至最大分辨率。
:::
